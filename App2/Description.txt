Desarrollo de aplicación para la gestión de eventos, permitiendo cierta interacción entre los 
usuarios como si de una red social se tratase. Los gestores podrán publicar sus eventos en la
aplicación de manera que los usuarios puedan conocer que eventos habrá disponibles, y el número
de asistentes a estos, permitiendoles indicar su propia asistencia. Esto le resultará de gran 
ayuda a los gestores para hacer una previsión de los recursos que necesitarán al conocer una 
aproximación de asistentes.
Para su funcionamiento deberá de iniciar la api desarrollada e introducir la dirección IP correcta.
